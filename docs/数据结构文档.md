# SoloCrop 数据结构文档

## 前端类型

### MediaAsset
- `id: string`
- `name: string`
- `file: File`
- `size: number`
- `durationSeconds: number`
- `width: number`
- `height: number`
- `createdAt: string`
- `lastUsedAt: string`

### TimelineTrackType
- `'video' | 'audio' | 'subtitle' | 'sticker'`

### TimelineTrack
- `id: string`
- `type: TimelineTrackType`
- `name: string`
- `locked: boolean`
- `muted: boolean`

### TimelineItem
- `id: string`
- `trackId: string`
- `mediaId?: string`
- `label: string`
- `startMs: number`
- `endMs: number`
- `sourceInMs: number`
- `sourceOutMs: number`
- `gain?: number`
- `speed?: number`
- `text?: string`
- `color?: string`
- `visualAdjust?: { brightness: number; contrast: number; saturation: number }`
- `transition?: { fadeInMs: number; fadeOutMs: number }`
- `stickerAdjust?: { xPct: number; yPct: number; scale: number; opacity: number }`
- `stickerStyle?: { variant: 'text' | 'pill' | 'badge' | 'burst'; bgColor: string; textColor: string; borderColor: string }`

### TimelineClip
- `id: string`
- `mediaId: string`
- `mediaName: string`
- `startSec: number`
- `endSec: number`
- `volume: number`
- `speed: number`
- `position: number`
- `visualAdjust?: { brightness: number; contrast: number; saturation: number }`
- `transition?: { fadeInMs: number; fadeOutMs: number }`

### TimelineProject
- `tracks: TimelineTrack[]`
- `items: TimelineItem[]`
- `durationMs: number`
- `fps: number`
- `aspectRatio: '16:9' | '9:16' | '1:1'`

### SubtitleSegment
- `id: string`
- `startMs: number`
- `endMs: number`
- `text: string`
- `confidence: number`
- `enabled: boolean`

### SubtitleSettings
- `model: 'tiny' | 'base'`
- `language: 'zh' | 'auto'`

### TemplatePreset
- `id: string`
- `name: string`
- `description: string`
- `source?: 'builtin' | 'custom'`
- `visualAdjust: Partial<{ brightness: number; contrast: number; saturation: number }>`
- `transition: Partial<{ fadeInMs: number; fadeOutMs: number }>`
- `stickers: Array<{ text: string; startOffsetMs: number; durationMs: number; adjust?: Partial<{ xPct: number; yPct: number; scale: number; opacity: number }>; style?: Partial<{ variant: 'text' | 'pill' | 'badge' | 'burst'; bgColor: string; textColor: string; borderColor: string }> }>`
- `subtitles: Array<{ text: string; startOffsetMs: number; durationMs: number }>`

### 导出相关类型
- `ExportPreset = 'source' | 'p720' | 'p1080'`
- `ExportMode = 'pieces' | 'merged' | 'final'`
- `ExportBitratePreset = 'auto' | '2000k' | '4000k' | '8000k'`

### ExportTaskItem（运行时队列）
- `id: string`
- `clips: TimelineClip[]`
- `pipelineVersion: number`
- `mode: ExportMode`
- `preset: ExportPreset`
- `bitrate: ExportBitratePreset`
- `retryCount: number`
- `maxRetries: number`
- `autoRetry: boolean`
- `retryable: boolean`
- `status: 'pending' | 'running' | 'success' | 'failed' | 'cancelled'`
- `progress: number`
- `current: number`
- `total: number`
- `message: string`
- `createdAt: string`

## IndexedDB Schema
数据库名：`solocrop`

### mediaAssets
索引：`id, name, createdAt, lastUsedAt`

### projectDrafts
索引：`id, updatedAt`

`content` JSON 结构（现行）：
- `projectId: string`
- `clips: TimelineClip[]`（兼容字段）
- `playheadSec: number`
- `selectedClipId: string | null`
- `exportPreset: ExportPreset`
- `exportMode: ExportMode`
- `exportBitrate: ExportBitratePreset`
- `timeline?: TimelineProject`
- `subtitleSettings?: SubtitleSettings`

## LocalStorage
### `solocrop.download.preference`
- `mode: 'browser' | 'directory'`
- `directoryName: string | null`

### `solocrop.editor.export.queue`
- `ExportQueueRecord[]`
- 刷新恢复时会将 `pending/running` 转为 `cancelled`

### `solocrop.editor.export.retry.pref`
- `autoRetryEnabled: boolean`
- `maxAutoRetries: number`

### `solocrop.template.custom.presets`
- `TemplatePreset[]`（仅保存 `source='custom'` 的本地自定义模板）

### `solocrop.subtitle.model.cache`
- `Array<'tiny'|'base'>`

## 需求矩阵与验收记录结构（文档层）
`docs/需求文档.md` 对比矩阵条目字段：
- `FeatureID`
- `功能名称`
- `CapCut基线描述`
- `SoloCrop当前状态`（`未开始/进行中/可用/已验收`）
- `缺失说明`
- `负责人`
- `目标里程碑`
- `DoD条目`
- `PlaywrightCaseID`
- `最近验收结果`（`PASS/FAIL`）
- `本轮必须项`（`是/否`）
- `备注`

变更记录字段：
- `日期`
- `变更内容`
- `对比矩阵变化`
- `验收报告`
