# SoloCrop 功能文档

## 产品定位
SoloCrop 是纯前端视频切割工具，运行于浏览器本地，无后端服务与数据库依赖。
功能对比与完成状态统一维护在 [需求文档.md](./需求文档.md)。

## 核心功能
1. 导入 MP4 视频素材（H264/AAC），并在素材库查看时长/分辨率/体积。
2. 自动按时长分段切割（快速关键帧 / 全精确切割）并导出下载。
3. 编辑器时间轴（首版）：
- 单视频轨、单音频轨、单字幕轨
- 拖拽排布、边缘裁剪、播放头联动
- 磁吸吸附（80ms 容差）
- 分割选中片段、删除、撤销/重做
4. 预览播放器：
- 按时间轴播放头进行素材预览
- 支持播放/暂停与时间同步
5. 自动字幕（本地 ASR 流程）：
- 模型档位：`tiny` / `base`
- 生成字幕段后可编辑文本、起止时间、启用状态、整体偏移
6. 滤镜与转场（首版）：
- 按片段设置亮度/对比度/饱和度
- 按片段设置淡入/淡出时长
- 预览窗口实时显示滤镜效果，导出成片时生效
7. 贴纸素材库（首版）：
- 内置文本贴纸快捷按钮（LIKE/WOW/NEW/SALE）
- 贴纸轨片段支持文本修改、位置、缩放、透明度调节
- 预览可见贴纸叠加，导出成片时烧录到视频
8. 模板系统（首版）：
- 内置 `Vlog 快剪` / `故事讲述` / `促销冲刺` 三套模板
- 一键应用模板后，批量写入滤镜、转场，并追加示例贴纸和字幕
- 模板作用于当前时间轴，不覆盖原素材与已有片段
9. 导出能力：
- 默认“成片导出（final）”
- 支持分辨率预设（源分辨率 / 720p / 1080p）
- 支持码率预设（自动 / 2Mbps / 4Mbps / 8Mbps）
- 支持字幕烧录（硬字幕）
10. 导出队列：
- 串行执行、取消当前任务、失败重试
- 自动重试（指数退避）
- 失败分类（可重试/不可重试）
- 队列历史持久化，刷新后恢复（running/pending -> cancelled）

## 需求闭环说明
- 每次新增完整功能必须同步更新 `需求文档.md` 的对比矩阵。
- 仅当 `DoD + Playwright 关键流 PASS` 后，功能状态才能标记为 `已验收`。

## 自动分段说明
- 输入分段秒数（正整数）。
- 示例：70 秒按 30 秒切割，目标段数为 3 段。
- 默认模式：快速关键帧切割。
  - 切点容差窗口：前后 1 秒。
  - 如果窗口内没有可用关键帧：跳到下一个关键帧。
  - 当前浏览器端采用关键帧近似估算策略，极少数素材会有额外偏差。
- 高级模式：全精确切割（重编码，速度较慢）。
- 执行阶段会先显示“初始化中/准备分段中”，再进入逐段导出进度，避免长时间显示 `0/0`。
- 若 FFmpeg 初始化超过 20 秒，会直接提示超时，避免界面长时间卡住无反馈。
- 支持设置生成视频输出目录：
  - 支持 File System Access API 的浏览器可选择目录，分段结果直接写入该目录。
  - 不支持或未授权时自动回退为浏览器默认下载方式。

## 限制
- 浏览器：Chrome/Edge 最新版。
- 大文件受浏览器内存影响，建议先按较短时长分段。
- 输入格式当前仅支持 MP4。
- 自动字幕为纯前端本地流程，首版以流程可用为目标，识别精度与设备性能相关。
